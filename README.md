Aurazone QA Assignment

Overview
This project contains basic Selenium automation scripts for testing the Aurazone website.

Tools Used
- Java
- Selenium WebDriver
- Chrome Browser

Setup Instructions
1. Install Java (JDK 8 or above)
2. Install Chrome browser
3. Add Selenium dependencies
4. Run the Java file

Test Scenarios Automated
- Open homepage
- Navigate to shop page
- Open product page
- Add to cart
- Basic navigation

Notes
- Simple XPath locators used
- Basic automation as per assignment requirement
